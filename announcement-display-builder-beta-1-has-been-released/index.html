Monday January the 12th 2026, the UI Suite team has proudly released the first beta of the Display Builder module, a display building tool for ambitious site builders:

A single display building tool deeply integrated with Drupal

Its powerful unified UI can be used instead of Layout Builder for entity view displays, Block Layout for page displays, and as a replacement of the Views' display building feature. No more struggle with overcomplicated inconsistent UIs!

Using the popular and battletested UI Patterns module as its engine, Display Builder can pull data from many Core & Contrib APIs: Content field, Menu, Breadcrumb, Views, and more.

Design system native

Display Builder module allows you to fully use your design system implementation (components, style utilities, icons, themes/modes, CSS variables...) directly in Drupal without the burden of compatibility layers:

  • UI Component, with Core's SDC
  • Icon packs, with Core's Icon API
  • Styles utilities, helpers & mode switchers, with the upcoming Core's Style API
  • CSS Variables, with the upcoming Core's API

Modern and fully customizable

A builder for the world of today, with the powerful features you would expect: dynamic previews, contextual menus, pattern presets, real-time collaboration…

The user interface is fully customizable, allowing the creation of different profiles for each of your business requirements, by easily adding and configuring Dynamic Islands plugins (powered by the Core’s HTMX API):

Streamline the marketing team experience while providing a powerful tool for your dev team!

Try it

The storage format is stable since beta1 so you are welcomed to start playing with the module. The more adventurous teams are already starting projects with it.

The installation is easy, it is “just” a normal Drupal module after all, and each integration (Entity View Display, Page Layout, & Views) is neatly implemented in its own sub-module:

$ composer require drupal/display_builder

$ drush en display_builder_entity_view display_builder_page_layout display_builder_views display_builder_ui

For more information on the release, the features and the roadmap, join us on Slack #display-builder

If you want to go further, the release of beta1 is the best moment to join the team. All talents are welcomed: developers, testers, designers… We are an international team of passionate drupalers, working in down-to-earth agencies, with a deep love for design systems and display building.

What’s next?

Beta 2 will follow in a few weeks with a commitment on API stability, full test coverage, a revolutionary translation mechanism, and the remaining little features we want to add to 1.0.0 scope.
